# desk-printer-client

1. Clone repo
2. Run `npm install`
3. Run `npm start`
4. Set **key** and **port** in **client.html** file as on app[``](https://)
5. Run client.html [live server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) or someting like that
6. Open in browser
7. Test print

# Build app

Run `npm run make`
